Scribbler

Rules

  1. Words must start and end with the letters provided.
  2. Words must contain at least 3 letters.
  3. No names or abbreviations allowed.
  4. No stems of base words (e.g. "limes" is invalid as its base word is "lime"; "hoarded" is invalid as its base word is "hoard").
  5. Scrabble rules! Each letter has a point value associated with it--the more uncommon the letter (e.g. "z" and "q"), the more points you'll get!
  6. If you can't think of a word, just guess! There are no penalties for guessing incorrectly, and if you happen to guess a word that exists, you might learn something.
